# Privacy Policy

_Last updated: 2025_

This Privacy Policy explains how **PDF Converter Bot**, operated by **PDF Converter Bot Team** (“we”, “us”, “our”), collects, uses, and protects information when you use the Telegram bot (“Bot”).

By using the Bot, you agree to this Privacy Policy.

---

## 1. Information We Collect

### 1.1 Telegram User Data  
Telegram automatically provides us with:

- User ID  
- Username (if available)  
- Language code  
- Chat ID  

This information is required for the Bot to function and communicate with you.

We **do not** collect names, phone numbers, or any other personal identifiers.

---

## 2. File Processing

When you send files (PDFs, images, documents), the Bot processes them solely to provide the selected feature:

- PDF conversion  
- Compression  
- OCR  
- Page editing  
- Watermarking  

Files are:

- Processed automatically by our server  
- Stored only temporarily  
- Deleted after task completion  
- Never manually reviewed  
- Never shared with third parties  
- Never used for training AI models

You are responsible for ensuring that the files you upload do not violate any laws or third-party rights.

---

## 3. Payments via Telegram Stars

PRO features may require activation using **Telegram Stars**.

- All transactions are handled entirely by Telegram.  
- We do **not** receive your payment card number, billing address, transaction metadata, or other sensitive payment information.  
- Telegram controls payment processing, refunds, and user balances.  
- We only receive a notification from Telegram that a purchase was completed.

Please review Telegram’s own policies for details on how Stars are managed.

---

## 4. Usage Statistics (Non-Personal)

The Bot may collect limited, anonymous technical data such as:

- Number of processed files  
- Feature usage (e.g., OCR count, compress count)  
- Error statistics  

This helps us improve performance but **cannot identify any individual user**.

We do **not** use analytics tools that track personal data.

---

## 5. Data Sharing

We **do not share** your data with:

- Third-party services  
- Advertisers  
- External analytics platforms  

We do not sell or trade user information.

---

## 6. Data Security

We use standard security measures to protect temporary file storage and communications.

However, no system is perfectly secure.  
You use the Bot at your own risk and should avoid uploading highly sensitive documents.

## 7. Children's Privacy

The Bot is not intended for children under 13.  
We do not knowingly collect personal data from minors.
